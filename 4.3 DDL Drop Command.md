## 1ï¸âƒ£ What is DROP Command?

The **`DROP` command** is a **DDL (Data Definition Language)** statement used to **permanently remove database objects** from SQL Server.

> DROP deletes the **object definition + its data**.

## 2ï¸âƒ£ Why DROP Command is Used?

DROP is used when:

* An object is no longer required
* Old / unused tables exist
* Cleaning test or temporary databases
* Replacing objects with new design
* Removing incorrect objects
