## 1Ô∏è‚É£ What is CREATE Command?

The **`CREATE` command** is a **DDL (Data Definition Language)** statement used to **create database objects** in SQL Server.

Once created:

* Objects are **stored permanently**
* Changes are **auto-committed**
* Objects exist until explicitly **ALTERed or DROPped**

---

## 2Ô∏è‚É£ Objects Created Using CREATE

In SQL Server (SSMS), `CREATE` can be used to create:

| Object Type | Purpose                  |
| ----------- | ------------------------ |
| DATABASE    | Store data               |
| TABLE       | Store records            |
| VIEW        | Virtual table            |
| PROCEDURE   | Stored logic             |
| FUNCTION    | Reusable calculation     |
| INDEX       | Performance optimization |
| SCHEMA      | Logical grouping         |
| SEQUENCE    | Generate numeric values  |
| TRIGGER     | Automatic action         |

## 5Ô∏è‚É£ CREATE TABLE from Another Table

## 8Ô∏è‚É£ CREATE FUNCTION

### Purpose

Returns a single value or table.

### Syntax

```sql
CREATE FUNCTION FunctionName (@param datatype)
RETURNS datatype
AS
BEGIN
    RETURN value
END;
```

### Example

```sql
CREATE FUNCTION CalculateGrade (@Marks INT)
RETURNS VARCHAR(10)
AS
BEGIN
    RETURN 
        CASE 
            WHEN @Marks >= 75 THEN 'Distinction'
            WHEN @Marks >= 60 THEN 'First Class'
            ELSE 'Pass'
        END
END;
```

### Use

```sql
SELECT dbo.CalculateGrade(82);
```
